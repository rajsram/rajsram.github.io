Online Reporting For HMS

This is an online reporting web responsive application for Hotels and restaurants. It Provides several reports and mail those reports as attachment. It contains dashboard and reports like room availability, guest info, collection and occupency in hotel and table matrix, daily sales, item sales and chart in restaurants.
